# Handover — セッション引き継ぎ

`.claude/context/HANDOVER.md` を生成または更新してください。次のセッションがすばやくコンテキストを復元できるようにします。

## 記載項目

以下のセクションを必ず含めてください:

### 1. Survey Progress（サーベイ進捗）
- 完了したフェーズ（scope / search / read / outline / draft / review / convert）
- 各フェーズの成果物ファイルと状態

### 2. Work Completed（完了した作業）
- 今セッションで完了した作業の詳細
- 新たに追加した論文、修正したセクション等

### 3. Decisions（判断）
- 構成に関する判断とその根拠
- 除外した論文やトピックとその理由

### 4. Unresolved Issues（未解決の問題）
- カバレッジの不足
- アクセスできなかった論文
- 判断保留の構成問題

### 5. Next Actions（次のアクション）
- 優先度順に次セッションでやるべきことを列挙
- 最初に実行すべきコマンド

### 6. Paper Statistics（論文統計）
- 収集済み論文数 / 分析済み論文数
- RQ 別カバレッジ状況

## 出力先

`.claude/context/HANDOVER.md` に書き出してください。既存の内容がある場合は最新セッション分で上書きしてください。
